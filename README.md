# DumbShot
> An elegant, painless one-click screenshot utility for Wayland (grim + slurp)

## Installing

### AUR (Arch User Repository)

DumbShot is available in AUR. Run this command:

```shell
yay -S dumbshot
```

Replace `yay` with your AUR helper (`paru`, `trizen` etc.)

### Compiling from source

Install dependencies:

- grim
- libnotify
- slurp
- wl-clipboard
- cargo (make) *
- rust (make) *
- wofi (optional: Recommended GUI for action selection ) *
- rofi ( optional: Alternative GUI ) *
- dmenu (optional: Minimalistic GUI) *
- hyprland (optional: To automatically determine monitors via hyprctl) *
- satty (optional: For editing screenshots) *

Run these commands:

```shell
git clone https://github.com/Faynot/dumbshot
cd dumbshot
make
```

## Using

Run `dumbshot` command in a terminal.
